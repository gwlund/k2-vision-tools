# Literature Review Example: Climate Adaptation Research

This example demonstrates a complete literature review workflow using the Citation & Source Verification skill with Zotero and Obsidian integration.

## Project Overview

**Research Question**: How have climate adaptation strategies evolved in urban coastal communities over the past decade?

**Timeline**: 4-week literature review
**Target**: 40-50 papers
**Output**: Systematic literature review for dissertation chapter

---

## Week 1: Initial Collection & Organization

### Day 1-2: Assess Existing Library

**Prompt**:
```
I'm starting a literature review on climate adaptation strategies in urban coastal communities.

First, let's see what I already have:
1. Search my Zotero library for papers on:
   - Climate adaptation
   - Urban resilience
   - Coastal communities
   - Sea level rise

2. For the papers found, show me:
   - How many total
   - Publication date range
   - Key authors
   - Any obvious gaps in coverage
```

**Expected Output**:
- List of 12-15 existing papers
- Identification of gaps (e.g., few papers on developing nations, mostly pre-2020)
- List of authors to search for more work

### Day 3-4: Create Organization Structure

**Prompt**:
```
Create a project structure for my climate adaptation literature review:

In Zotero:
- Create collection: "Dissertation - Climate Adaptation"
- Create sub-collections:
  - "01 - Foundational Theory"
  - "02 - Urban Case Studies"
  - "03 - Coastal Specific"
  - "04 - Methodology Papers"
  - "05 - Recent Reviews (2020+)"

In Obsidian:
- Create folder: Research/Climate-Adaptation/
- Create subfolders:
  - Literature/
  - Synthesis/
  - Quotes/
- Set up literature note template with tags for:
  - Geographic focus
  - Adaptation type
  - Methodology
  - Year
```

### Day 5-7: Expand Collection

**Prompt**:
```
I need to expand my collection to 40-50 papers. Help me identify:

1. Seminal papers (pre-2015):
   - Papers on urban climate adaptation that are foundational
   - Most-cited papers in this area
   - Key theoretical frameworks

2. Recent work (2020-2024):
   - Recent systematic reviews
   - Novel adaptation strategies
   - Post-COVID perspectives

3. Methodological diversity:
   - Quantitative studies
   - Qualitative case studies
   - Mixed methods
   - Modeling studies

For each suggestion, provide:
- Why it's important
- Expected DOI or search terms to find it
- Which Zotero sub-collection it belongs in
```

**Expected Outcome**:
- List of 25-30 papers to add
- Mix of old/new, methods, and geographic coverage
- Clear rationale for each inclusion

## Week 2: Annotation & Quality Assessment

### Day 1-3: Create Literature Notes

**Prompt (for each paper)**:
```
I just read this paper: "[Paper Title]" by [Authors] ([Year])

Create a comprehensive Obsidian literature note:

1. Use the literature note template
2. Zotero key: [KEY]
3. Quick summary: [2-3 sentences in my own words]
4. Key findings: [3-5 bullet points]
5. Methodology: [brief description]
6. Geographic focus: [location]
7. Adaptation strategies discussed: [list]
8. Strengths: [2-3 points]
9. Limitations: [2-3 points]
10. Relevance: How this fits into my dissertation chapter on urban coastal adaptation
11. Key quotes: [2-3 important quotes with page numbers]
12. Tags: Add appropriate tags based on content
13. Rating: [1-5 stars based on quality and relevance]

Link to related papers I've already annotated if applicable.
```

### Day 4-5: Verify Key Sources

**Prompt**:
```
These are the 10 most important papers for my literature review:
1. [Paper 1]
2. [Paper 2]
...
10. [Paper 10]

For each, run the 5-Point Verification Protocol:
1. Authority - Check author credentials
2. Currency - Are there more recent papers that update/supersede this?
3. Accuracy - Cross-reference key claims with other sources
4. Purpose - Identify funding sources, potential biases
5. Coverage - Is treatment comprehensive?

Create a verification report for each with a final recommendation:
- ✅ Cite with confidence
- ⚠️ Cite with caveats (specify what caveats)
- ❌ Do not cite (explain why)
```

## Week 3: Synthesis & Pattern Analysis

### Day 1-2: Thematic Analysis

**Prompt**:
```
Analyze all 45 papers in my "Climate Adaptation" Zotero collection.

Identify major themes:
1. What are the 5-7 major themes discussed across papers?
2. How many papers address each theme?
3. Which themes have consensus vs. debate?
4. Which themes are under-researched?

For each theme:
- Define it clearly
- List key papers (with Zotero keys)
- Summarize state of knowledge
- Note any contradictions or debates
- Identify gaps

Create an Obsidian synthesis note for each theme linking to relevant literature notes.
```

### Day 3-4: Temporal Evolution Analysis

**Prompt**:
```
Analyze how thinking on urban coastal climate adaptation has evolved:

1. Early period (2010-2015):
   - What were the dominant ideas?
   - Key papers that established the field
   - Initial frameworks and approaches

2. Middle period (2016-2019):
   - How did thinking shift?
   - New methodologies introduced
   - Emergence of which concepts?

3. Recent period (2020-2024):
   - Current state of the art
   - Influence of COVID-19
   - Emerging trends
   - Future directions

Create a timeline visualization showing:
- Key papers at each period
- Evolution of major concepts
- Methodological advances
- Shifts in geographic focus
```

### Day 5: Citation Network Analysis

**Prompt**:
```
Analyze citation patterns in my collection:

1. Which papers in my collection are most cited by other papers in the collection?
   - Identify the 5-10 most influential papers
   - These are my "foundational" papers

2. Are there papers frequently cited BY my collection that I don't have?
   - These are gaps I should fill
   - Prioritize top 5 missing papers

3. Identify citation clusters:
   - Groups of papers that heavily cite each other
   - These represent sub-communities or approaches

4. Find the "bridges":
   - Papers that connect different clusters
   - These are good for synthesis

Create an Obsidian map showing these relationships.
```

## Week 4: Writing & Final Verification

### Day 1-2: Create Outline

**Prompt**:
```
Based on my 45 papers and synthesis notes, create a detailed outline for my literature review chapter:

Structure:
1. Introduction
   - Scope and importance
   - Research question
   - Organization of review

2. Historical Development [Use temporal analysis from Week 3]
   - Early work (2010-2015)
   - Evolution (2016-2019)
   - Current state (2020-2024)

3. Thematic Analysis [Use themes from Week 3]
   For each theme:
   - Overview
   - Key studies (cite specific papers)
   - Current consensus or debates
   - Representative quotes

4. Methodological Approaches
   - Quantitative studies and findings
   - Qualitative insights
   - Modeling approaches
   - Mixed methods contributions

5. Geographic Patterns
   - Developed vs. developing nations
   - Different coastal contexts
   - Transferability of strategies

6. Synthesis & Gaps
   - What we know
   - What remains unclear
   - Methodological limitations
   - Future research needs

For each section, suggest:
- Which papers to cite (from my collection)
- Key points to make
- Representative quotes
- Connections between papers
```

### Day 3-4: Draft & Cite

**Prompt (iterative)**:
```
I'm writing the section on [Theme/Topic].

Help me:
1. Draft 2-3 paragraphs synthesizing what papers say about this
2. Ensure proper citations in APA format
3. Balance coverage (don't over-rely on one paper)
4. Include both consensus findings and debated points
5. Use direct quotes sparingly but effectively
6. Transition smoothly between papers/ideas

Papers to include:
- [Paper 1] - Main contribution: [brief note]
- [Paper 2] - Main contribution: [brief note]
...

Style: Academic but accessible, dissertation-level writing
```

### Day 5: Final Verification Pass

**Prompt**:
```
Final verification before submitting literature review chapter:

1. Citation Check:
   - List every citation in my draft
   - Verify each has complete entry in Zotero
   - Check for formatting consistency
   - Ensure no "orphan" citations (cited but not in bibliography)

2. Claim Verification:
   - Extract all factual claims
   - Verify each has appropriate citation
   - Double-check any statistics or specific numbers
   - Confirm I'm representing sources accurately

3. Coverage Check:
   - Did I cite all major papers in my collection?
   - If not, explain why some were excluded
   - Are citations balanced across time periods?
   - Am I over-relying on any single source?

4. Quality Check:
   - Verify no retracted papers
   - Check if any key papers have been updated/corrected
   - Confirm recent literature search caught latest work

5. Academic Integrity:
   - All direct quotes in quotation marks with page numbers
   - Paraphrasing substantially different from originals
   - No close paraphrasing that could be plagiarism
   - Proper attribution of ideas

Generate a final verification report with any issues found.
```

## Lessons Learned

### What Worked Well
- Creating Zotero sub-collections early helped organization
- Literature note template ensured consistency
- Weekly synthesis prevented getting overwhelmed
- 5-Point Verification caught two questionable sources early
- Citation network analysis revealed gaps I wouldn't have found otherwise

### Challenges Encountered
- Some papers hard to categorize (solved by using multiple tags)
- Zotero metadata sometimes incomplete (manual cleaning required)
- Balancing depth vs. breadth (used rating system to prioritize)
- Some contradictory findings required careful synthesis

### Time Breakdown
- Collection/organization: 8 hours
- Reading and annotating: 35 hours (45 papers × ~45 min average)
- Synthesis and analysis: 10 hours
- Writing: 12 hours
- Verification and editing: 5 hours
- **Total**: ~70 hours over 4 weeks

### Tips for Next Time
1. Start with recent review papers to quickly identify key older papers
2. Use "snowball" method: follow citations from key papers
3. Set reading goals: 2-3 papers per day sustainable
4. Annotate immediately after reading (details fade fast)
5. Weekly synthesis crucial for seeing patterns
6. Build bibliography as you go (don't wait until end)
