# K2 Vision Tools - Project Instructions

## Repository Purpose

Setup automation and development tools for K2 Vision team members. Contains the MacBook setup script, Zotero CLI, and Obsidian vault setup.

## Key Directories

| Directory | Purpose |
|-----------|---------|
| `scripts/setup-macbook.sh` | Automated MacBook setup for new team members |
| `scripts/zotero-cli/` | 25 Python CLI commands for Zotero API access |
| `scripts/obsidian-setup/` | Obsidian vault provisioning with Zotero integration |
| `docs/` | Setup guide and documentation |
